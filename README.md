# Zenny Estates - Luxury Real Estate Platform

A premium, modern, and highly luxurious real estate website focused on showcasing high-end properties and luxury buildings. Built with a "Platinum Silver & Midnight Navy" theme.

## 🚀 LIVE DEPLOYMENT

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fzainabdigital1020-dev%2Freal-estate)

## ✨ Features

- **Ethereal Midnight & Platinum Theme**: A sophisticated architectural aesthetic.
- **Advanced Filtering**: Filter by Location, Property Type, Price, Beds, and Purpose (Buy/Lease).
- **Interactive Map View**: Full-screen Google Maps integration for property search.
- **Glassmorphism Design**: High-end UI components with translucent effects.
- **Responsive Layout**: Optimized for Desktop, Tablet, and Mobile.
- **Direct Engagement**: Floating WhatsApp concierge button.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React & React Icons
- **Styling**: Vanilla CSS with Modern Variables

## 📥 Local Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/zainabdigital1020-dev/real-estate.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
